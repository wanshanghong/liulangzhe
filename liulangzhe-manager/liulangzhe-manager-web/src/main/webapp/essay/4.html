法国斗牛犬可以顺产吗

如何解决法国斗牛犬难产的问题

狗难产情况一：阵痛微弱。某些母犬由于生产时用力不足，或是阵痛幵始时才微微用力，再加上平曰运动不足，筋骨缺乏锻炼，就会造成阵痛微弱。有的母犬怕痛、肥胖，或对生产的环境不满，也容易引起阵痛微弱。另一方面阵痛微弱也可能是由于母犬血液中所含钙质过低，或是由于引发阵痛的荷尔蒙不活性所引起的。

狗狗难产情况二：产道狭窄。胎儿并不过大，但却无法顺利的娩出，此时极可能是母犬本身的骨盆狭小，使胎儿无法通过，或阴道发育不全或狭小所引起。若有此种情形，请兽医实行剖腹产。

狗狗难产情况三：胎位不正。生产时，胎儿不照正常情况以头部先娩出，而是采取其他不正确的姿势往产道前进，我们称之为胎位不正。有些不正确的胎位仍然可以顺利生产，有些却成为决定性的难产，因此若母犬阵痛强烈，而时间也紧迫，却仍然产不下胎儿时，就要怀疑是否胎位不正了。通常胎位不正可通过X光照射而诊断出来。
