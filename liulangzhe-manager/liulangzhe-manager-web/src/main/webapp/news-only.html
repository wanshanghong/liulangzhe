今日英国发生弃养狗事件 怀孕斗牛犬被遗弃公园
更新时间: 2019-03-12

佛罗伦斯在公园被发现时，没人知道牠是什么狗

喜欢品种狗是一回事，但是为了尊重生命活着的权利，请务必选择合格的繁殖场，以免让这种恶质繁殖商人越来越多!英国最近发生弃养狗事件，有伦敦市民在一处公园裡的小巷，发现一隻狗被人关在笼子裡，整笼丢在那裡。

当时狗狗的状况很差，全身沾染了粪便，完全看不出牠到底是什么狗、是什么颜色，等到动保团体RSPCA到场，才知道牠是一隻白色的斗牛犬，肚子隆起、显然怀有身孕。但等到他们把狗狗送医，才知道都来不及了。

兽医研判牠是一隻已经生产过很多次的繁殖犬。

这隻狗狗被取了很美的名字叫「佛罗伦斯」，兽医为牠做超音波检查时，发现牠肚子裡有个宝宝，但已经去世了，他们观察到佛罗伦斯有多次生产的迹象，应该是一隻繁殖犬，前主人很可能因为牠生到难产，懒得送医就将牠随意丢弃在公园。

不只如此，因为发现的地方太过隐密，RSPCA怀疑弃养牠的人就是当地居民，因为那是一隻个只有当地人才知道的小巷子，他们还在追查到底是谁抛弃了佛罗伦斯，让牠差点和肚子裡的宝宝一起静静死去。
